# mvs-geometry-tool
Some useful tools for multi-view stereo


## Run on MacOS

Assume Python@3.11:

- tkinter
  - brew install python-tk@3.11
  - pip install tk

Other requirements:

```
opencv-python
Pillow
matplotlib>=3.5.1
OpenEXR==1.3.8 #(probabily require OpenEXR==1.3.8, if you see ImportError: undefined symbol: _ZTIN13IlmThread_3_14TaskE;)
imageio
```
